Web Design Fundamentals

This course introduces students to the principles and techniques of effective web design. Through a combination of theory and hands-on practice, students will learn to create visually appealing, user-friendly websites using industry-standard design tools and technologies. Topics covered include layout design, typography, color theory, responsive design, and user experience (UX) principles.

Learning Objectives:
->Understand the fundamentals of web design including layout, typography, and color theory.
->Develop proficiency in using design software such as Adobe XD, Sketch, or Figma.
->Apply responsive design techniques to ensure websites look great on all devices.
->Incorporate usability and accessibility principles into web design.
->Create wireframes, mockups, and prototypes for web projects.
->Gain an understanding of HTML and CSS for implementing designs.

Topics Covered:
1.Introduction to Web Design Principles
2.Understanding User Experience (UX) and User Interface (UI) Design
3.Fundamentals of Typography and Color Theory
4.Layout Design and Composition
5.Design Tools and Software (e.g., Adobe XD, Sketch, Figma)
6.Responsive Web Design with CSS Media Queries
7.Usability and Accessibility in Web Design
8.Prototyping and Wireframing
9.Introduction to HTML and CSS for Designers
10.Designing for Different Platforms (Desktop, Mobile, Tablet)

Prerequisites:
No prior experience in web design is required. Basic computer literacy and familiarity with using the internet are recommended.

Assessment and Grading:
Assessment will be based on design projects, assignments, critiques, and a final portfolio showcasing the student's design skills and creativity.

Course Materials:
*Online resources, tutorials, and readings on web design principles and techniques
*Design software and tools (free or trial versions)
*Access to design libraries, stock images, and icons
